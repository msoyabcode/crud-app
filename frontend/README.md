# MERN CRUD App

## 🚀 Features
- User Authentication (JWT + Cookies)
- Protected Routes
- Add, Edit, Delete Tasks
- Multiple Delete Functionality
- Secure Backend APIs

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## 🔐 Authentication
- JWT Token based authentication
- Cookie-based session handling
- Middleware for protected routes

## ⚙️ Setup
1. Clone the repo
2. Install dependencies:
   npm install
3. Start backend:
   nodemon index.js
4. Start frontend:
   npm run dev
