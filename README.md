# 🚀 MERN CRUD App

A full-stack CRUD application built using MongoDB, Express, React, and Node.js with JWT Authentication.

## 🔐 Features
- User Authentication (JWT + Cookies)
- Protected Routes
- Add, Edit, Delete Tasks
- Multiple Delete Functionality

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## ⚙️ Setup
1. Clone repo
2. Run backend: nodemon index.js
3. Run frontend: npm run dev

## 📂 Project Structure
- /frontend → React App
- /backend → Node + Express API
